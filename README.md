# Astro with Phaser 3 and Svelte

## Initialize the project

```sh
pnpm create astro --template with-tailwindcss
cd astro-phaser-svelte-starter
pnpm astro add svelte
pnpm add -D prettier prettier-plugin-astro
pnpm add phaser
```
